# 香港居 - 劉以鬯

本倉庫收錄了香港文學泰斗**劉以鬯**的長篇小說作品《香港居》。

## 內容簡介

《香港居》於1960年7月至12月連載於《星島晚報》，是劉以鬯現實主義文學的代表作之一。小說以第一人稱敘述，描寫一名從南洋（星馬地區）回流香港的職業作家，帶著妻女在擁擠的都市中尋找棲身之所的艱辛歷程。

故事深入展現了1960年代香港的住房危機。主角一家先是租住舊同學潘承富的梗房，卻捲入潘家虛偽的婚姻與經濟糾紛中。隨後，他們經歷了多次搬遷，與各式各樣的人物（如試圖轉租牟利的周女士、崇尚西方文化的莎莉等）同處一室。小說透過主角這位「文化勞工」的雙眼，冷靜而細膩地觀察了分租房（梗房）內的摩擦、貪婪與人性百態。故事最終以主角一家寄望遷入新落成的「模範屋」作結，反映了當時香港從私樓分租轉向公共房屋建設的時代變遷。

## 核心主題

*   **住房危機：** 探討在人口稠密的都市中「居住」的尊嚴與困境。
*   **人性與生存：** 描寫小市民在生存壓力下表現出的自私、偽善與掙扎。
*   **階級與異鄉感：** 文人身分與經濟現實的落差，以及回流者對故地的陌生感。
*   **新舊交替：** 透過對歐美流行文化（如貓王、爵士樂）的描寫，展現1960年代香港青年的現代化轉向。

## 關於作者

**劉以鬯**（1918－2018），原名劉同騰，香港著名小說家、編輯。他是中國現代意識流小說的開拓者，代表作包括《酒徒》、《對倒》（後者啟發了王家衛電影《花樣年華》）。

劉以鬯的創作跨越「嚴肅文學」與「流行文學」，《香港居》雖為報章連載的「通俗」小說，卻展現了極強的社會觀察力與寫實功力，是研究1960年代香港社會史與文學史的重要文本。
